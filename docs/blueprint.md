# **App Name**: Questify

## Core Features:

- Gamified Checklist: Create and manage tasks with associated XP and HP rewards.
- Character Card: Display user's XP, HP, level, and other stats on a virtual character card.
- XP and HP Calculation: Automatically calculate XP and HP gains based on completed tasks.
- Reward System: Award daily, weekly, and monthly prizes upon reaching certain XP/HP thresholds.
- Task Scheduling: Allow users to schedule tasks with deadlines.
- Motivational Messages: Use an AI tool to provide dynamic, motivational messages based on task progress.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to invoke a sense of magic and adventure.
- Background color: Light gray (#F0F0F0) for a clean, modern feel.
- Accent color: Vibrant orange (#FF9800) for interactive elements and rewards.
- Body and headline font: 'Inter', a sans-serif font, provides a modern and neutral look for both headlines and body text.
- Use custom, game-like icons for tasks, rewards, and stats.
- Card-based layout for easy task management and progress tracking.
- Use subtle animations and transitions to provide feedback and celebrate task completion.
